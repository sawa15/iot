Система умного капельного полива с использованием централизованного мониторингового центра

Регулярный полив – залог хорошего урожая.
Моя система позволяет эффективно орошать посаженные культуры.
С помощью системы можно экономить денежные ресурсы за счёт сокращения использования воды,чтобы получить максимально обильный урожай.
Использование энергоэффективных компонентов позволяют автономно использовать устройство на протяжении полугода.
Для связи с мониторинговым центром устройства можно подключать как по кабелю, так и с использоваением wi-fi-модуля.
Конечные устройства передают в мониторинговый центр свой идентификационный номер и информацию с датчика влажности, которую можно проанализировать.
Устройство в вашей грядке подклчается напрямую к сети капельного полива и с помощью специальных клапанов начинает подавать воду для полива до тех пор, пока влажность почвы не станет достаточной для питания ваших растений.
Мониторинговый центр позволяет использовать до девяноста девяти устройств одновременно, позволяя выводить график влажности почвы в реальном времени, для более удобного отслеживания ситуации на частных насаждениях сельскохозяйственных культур.
Интерфейс с панелью отслеживания статуса всех подключённых устройств является открытым и позволяет увидеть изменение влажности своих грядок, не запоминая логинов и паролей для доступа.
